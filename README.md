# Wraithgrove

Mobile arena ARPG — top-down auto-attack survival with character-RPG meta layer, idle base, gear gacha, async PvP.

**Live:** https://thehivemakes.github.io/wraithgrove/

Vanilla JS, zero frameworks, ~4,500 LOC across 38+ modules. iOS/Android wrap via Capacitor 5 (separate deploy). Server endpoints via Cloudflare Worker (separate deploy).

## Status

Procedural-art prototype for clone-fidelity validation. Real art (43 Midjourney briefs in `art/`) and audio (manifest in `audio/MANIFEST.md`) wire in once assets land on disk.
